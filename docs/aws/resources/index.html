AwsEc2InstanceAmazon Elastic Compute Cloud (Amazon EC2) is a web service that provides secure, resizable compute capacity in the cloud. It is designed to make web-scale cloud computing easier for developers. Amazon EC2’s simple web service interface allows you to obtain and configure capacity with minimal friction.Goto Docs
AwsEc2KeyPairA key pair, consisting of a private key and a public key, is a set of security credentials that you use to prove your identity when connecting to an EC2 instance.Goto Docs
AwsEc2SnapshotAmazon EBS provides the ability to create snapshots (backups) of any EBS volume. A snapshot takes a copy of the EBS volume and places it in Amazon S3, where it is stored redundantly in multiple Availability Zones.Goto Docs
AwsEc2VolumeAn Amazon EBS volume is a durable, block-level storage device that you can attach to your instances. After you attach a volume to an instance, you can use it as you would use a physical hard drive.Goto Docs
AwsElasticIpAn Elastic IP address is a static IPv4 address designed for dynamic cloud computing. you can mask the failure of an instance or software by rapidly remapping the address to another instance in your account. Alternatively, you can specify the Elastic IP address in a DNS record for your domain, so that your domain points to your instance.Goto Docs
AwsInternetGatewayInternet gateway are horizontally scaled, redundant, and highly available VPC components that allows communication between your VPC and the internet.Goto Docs
AwsNatGatewayNetwork address translation (NAT) gateways enable instances in a private subnet to connect to the internet or other AWS services, but prevent the internet from initiating a connection with those instances.Goto Docs
AwsNetworkInterfaceAn elastic network interface is a logical networking component in a VPC that represents a virtual network card.Goto Docs
AwsPrefixListA prefix list is a set of one or more CIDR blocks. You can use prefix lists to make it easier to configure and maintain your security groups and route tables.Goto Docs
AwsReservedInstanceReserved Instances (RI) provide a significant discount (up to 72%) compared to On-Demand pricing for EC2 Instances and provide a capacity reservation when used in a specific Availability Zone.Goto Docs
AwsRouteTableA route table contains a set of rules, called routes, that are used to determine where network traffic from your subnet or gateway is directed.Goto Docs
AwsSecurityGroupA security group acts as a virtual firewall for your instance to control inbound and outbound traffic. A security group specifies the actions that are allowed, not the actions that are blocked.Goto Docs
AwsTransitGatewayAWS Transit Gateway connects VPCs and on-premises networks through a central hub. This simplifies your network and puts an end to complex peering relationships. It acts as a cloud router – each new connection is only made once.Goto Docs
AwsVpcAmazon Virtual Private Cloud (Amazon VPC) is a service that lets you launch AWS resources in a logically isolated virtual network that you define. You have complete control over your virtual networking environment, including selection of your own IP address range, creation of subnets, and configuration of route tables and network gateways. You can use both IPv4 and IPv6 for most resources in your virtual private cloud, helping to ensure secure and easy access to resources and applications.Goto Docs
